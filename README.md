Gesture Flow

Gesture Flow is a mobile application built to support learning Malaysian Sign Language, BIM. You built the app to reduce communication gaps between Deaf and hearing communities by offering interactive lessons and real time hand sign practice through the camera. The app focuses on active learning instead of passive videos. Users practice signs, receive instant accuracy feedback, and track progress inside a simple mobile interface.

Key Features
- Structured BIM lessons from basic to advanced
- Real time hand gesture recognition using camera input
- Instant feedback on gesture accuracy
- Gamified quizzes and learning progress tracking
- Firebase based authentication and user data storage

Built With
-Flutter and Dart
-Firebase Authentication and Firestore
-Teachable Machine model
-TensorFlow Lite for on device gesture recognition
